# MISHARP SEO OS V2

MISHARP SEO OS는 자사몰, 스마트스토어, 경쟁사 상품 URL을 분석하여  
카페24 입력용 SEO와 콘텐츠 SEO 참고 항목을 함께 생성하는 Streamlit 앱입니다.

## 주요 기능

- 상품 URL 1개 SEO 생성
- 상품 URL 여러개 일괄 SEO 생성
- 카테고리/목록 URL에서 상품 링크 자동 수집
- 카페24 입력용 SEO 생성
- 콘텐츠 SEO용 slug / H1 / 롱테일 키워드 / SEO 점수 / 블로그 문장 생성
- CSV 다운로드

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
